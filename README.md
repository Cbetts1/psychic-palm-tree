# AURa OS

AURa is an AI-native operating environment.

- Thin client on the device (Android/Termux/Linux)
- Virtual OS runtime in the cloud (VHost)
- Cloud workers for AI generation (images, video, APKs, docs, audio)
- Virtual filesystem mounted from cloud storage
- Operator-grade shell and menu interface

This repo starts from zero and will be built layer by layer.
